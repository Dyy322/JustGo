# 即行 JustGo

> 发现城市的精彩，更创造属于你的时刻。

## 概述

即行（JustGo）是一款以**时效性**著称的、**找搭子**为主要功能的平台。收集各类活动信息，接入地图 API，在主页显示附近且最近即将到来的热点活动，能够在具体活动下发布找搭子帖子或开房间，找到搭子后按规定时间地点赴约，说走就走。

## 四大核心亮点

### 📡 城市活动雷达 —— 再也不错过

实时打捞散落在城市角落的精彩：小众独立书店分享会、突然官宣的周末市集、公园里限定一周的赏花会、新开的人气展览……所有值得去的、好玩的活动，不再是你的信息盲区。

### 👥 从发现到搭子，一键即约

看到心动的活动，却找不到人一起？在活动页面点击 **"征个搭子"**，迅速找到兴趣相投、时间匹配的伙伴。看展、看演出、逛市集，和搭子一起分享快乐，体验翻倍。

### 🧩 自由发起场 —— 你也是活动发起人

不只跟随，更能创造。突然想打场球、约人吃宵夜、组队爬山、探店打卡……在自由发起场，随时为任何想法发起邀约，找到愿意一起实现的同伴。你不仅是参与者，更是自己生活的策划人。

### ✨ 智能游玩计划 —— 为你量身推荐

根据你的兴趣、过往参与和搭子匹配情况，即行会为你定制个性化的活动推荐与一日/半日游玩动线。无论是文艺路线、美食探索，还是运动社交，我们都懂你想要的"恰好适合"。
